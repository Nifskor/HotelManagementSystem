### 현재 알려진 버그 
>1. 아이디 생성까지 문제가없지만 (staff) / 아이디생성후 -> 다시 (직원) 아이디 비밀번호 변경후 새로생성시 널문자 오류로 인한 오류 발생 
>2. 체크인할때 검색하는 기능이 잘안되는부분 (위에꺼는 잘표시가 되지만 아래부분 데이터가 간혈적으로 잘표시되지 않는 문제발생)
>3. 관리자메뉴 호실변경시 간혈적으로 제대로 업데이트가 되지않는 상황이 발생
